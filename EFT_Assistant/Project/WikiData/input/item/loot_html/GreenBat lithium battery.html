CENTER:&ref(./GreenBat リチウムバッテリー.jpg,500x500);
*概要 [#a80fde4c]
3.7V公称3400mAhのリチウムイオンバッテリー。ライトやその他の工業製品に使用する。
リチウムイオン18650電池。

#fold(&color(Black){小ネタ};){{
モバイルバッテリーなどに広く使われる「18650」という種類のバッテリーセル。
直径18mm×長さ65mmの規格が名前の由来


容量が大きく長時間使用できる上、再充電が可能で電圧も高いため利便性が高いが、保護回路などが必要になる。
単3電池などと違い大抵はボックス型にまとめられ、金属リボンでそれぞれをスポット溶接して回路と繋ぐ形で使用されることが多い。
