      Dogtagのソース

      これらのキーワードがハイライトされています：
        *Dogtag USEC・BEAR/ ドッグタグ [#z3e9b98f]
CENTER:&ref(img/Dogtag.gif,);
*概要 [#e0035447]
認識票。死亡した兵士の身元確認に使用される。
詳細を開くと、持ち主のプレイヤー名、所属、レベル、死亡した年月日、殺害したプレイヤー名、最後の一撃となった武器の名前が閲覧できる。

プレイヤーPMCの死体のアイテム欄、ドックタグスロットに必ずスポーンする。倒した後でないと表示されない。
調べると倒された時の状況が記されており、持ち帰ることで「倒した相手」の表に名前「？？？？」から変わり、他SCAVを倒した時と同じように最後の一撃の部位が見られる。最後にインベントリに入れて持ち帰らないと「？？？？」のままになる。

その倒されたプレイヤーのレベルが高いほど売買価値も高い。40レベルで2万ルーブル弱まで行くが、逆に低レベルのものは売値も安く、公式wikiでもLv.20程度かそれ以上を集めた方がいいと書いている。

トレーダーのタスクで渡す必要がある他、多様な取引で使用できるのでPMCを倒した場合は可能な限り持ち帰りたい。（開発が安直に「金で買えないもの」を増やしているため。）
USECとBEARでは別アイテムのため、引き渡し時には注意。

ケース類の中では、[[Dogtag case]] 、[[S I C C organizational pouch]] 、[[Lucky Scav Junk box]] に収納可能。
現在、フリーマーケットには出品できない。


#fold(&color(Black){小ネタ};){{
ロシア語では「Армейский жетон 」で直訳は「軍のトークン」という意味。
全体がアルミニウム製で、同じくアルミのボールチェーンに通される。
USECの物は縁を折り返しており、BEARの物の方が造りが単純。

BEARの中央にある線は単純に読みやすさを補う強調線のようなものかと思われる。（国によって、1枚をその縦横いずれかに折って千切るための切れ込みを入れることもあるが、そのように使う記述は見当たらない。）

}}

なし

''その他''
-死亡したPMC（プレイヤー）のドッグタグのスロットに装備（装備）されている。NPCSCAVにはスロットはあるが湧くことはない。
生きている間に自分の装備欄から見えず、取り外すことも表示もされない。
これ以外には入手方法は無い。（他人の死体から取り外したものを持ち歩いている場合は別）
