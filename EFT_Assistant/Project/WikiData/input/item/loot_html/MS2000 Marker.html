      MS2000 Markerのソース

      これらのキーワードがハイライトされています：
        *MS2000 Marker [#pe6ccaa1]
CENTER:&ref(img/MS2000 marker.jpg,nolink,500x500);
&color(Gray){画像はオブジェクトモデルを上下反転};
*概要 [#tc29d684]
フラッシュストロボライト。

主にタスクで乗り物や所定の場所に設置し消費する。

銃撃して破壊できるようなので、設置時に敵対プレイヤーを近づけないのは勿論、いたずらに発砲しないようにしよう。

一部タスクでは（チュートリアル的に）事前に配布されるが、それ以降は購入が必須になる。


#region(小ネタ)


自分の位置や目標物を味方に知らせるためのマーカー。
ヘルメットなどに括りつけたり、入れたポーチから発光部を出して使用する。
''ゲーム内ではこの機能は利用できない。''

夜間に設置し、NVGを装着していると発光しているのを確認できる。



ACR ELECTRONICS社で開発された。
https://www.acrartex.com/products/ms-2000-m2-strobe
http://www.ostrich.co.jp/wps/wp-content/uploads/2016/11/manual-y1-03-0275b.pdf

外装を上に引き出せばハンドライトのように光が集中し、位置を四方八方に暴露させずに、味方航空機だけへ合図を送りやすくなる。

暗い色のカバーは白色光を遮断し、赤外線のみを透過させるので、ナイト・サーマルビジョンでのみ感知できるようになる。
内蔵された青色のフィルターで色を変えて、敵味方を区別できる。（ストロボの規則的な光が発砲炎に見え、誤射が生じやすいため。）
#endregion

*タスクでの使用 [#zfdfc727]
-''ゲーム内ヒント''
--''Special Slotsを活用しよう！''
マーカーは''「[[ポーチ]]」''以外にも''「Special Slots」（ポケット右横の3マス枠）''に%%%入れることが出来ます。%%%
この枠やポーチに入れているアイテムは死亡時にロストしないので、レイドに持ち出す際はそこに入れておくと、%%%万が一レイド中に死亡してもマーカー分の損害を防ぐことが出来ます。%%%
-''マーカーに関する注意点''
--マーカーの設置はインタラクトキー（F）を押すことで出来る。（間違って左クリックしない事！銃を持ってたら発砲します！）
--設置後は数十秒のカウントダウンが始まるので、カウントが終わるまで生存する必要がある。（設置後は安全な場所で待機しよう！）
---タスクによっては設置後の待機時間（カウントダウン）が違う事があるので、オフラインなどで事前に確認しておく良いでしょう。（10秒、30秒、40秒などのカウントがある）
---設置後は目標物にマーカーがペタッと付きカウントが始まりますが、マーカーに視点を合わせると再取得する事も出来ます。
再取得した場合はインベントリに戻ってきますが、カウントダウンが終了しても設置扱いになりません。（設置がやり直しになる）
--1つの目標物に対してマーカーを複数設置できる事がありますが、カウントダウン終了後は設置したマーカーは全て消失します。（失ったマーカー分の費用が掛かります）
---設置した目標物に対しては設置が完了扱いになりますが、バグによって設置扱いにならない場合があるので、基本的には１目標１マーカーに留めておきましょう。
--&color(red,){カウントダウン中に''死亡（KIA,MIA,脱出など）すると%%%やり直しになる。''%%%};（設置したマーカーは戻ってきません！）
