*概要 [#e0035447]
プリント回路板。
微細な電子部品の集合体。電子工作には不可欠なリソース。
#region(小ネタ)
アイテム画像は切り欠きの位置などから現行の「PCIe」規格の前世代である「PCI」規格バススロットに装着するカードであると思われる。（32bitの5ボルト専用？）
1990年代後半から2000年代初頭に普及したがすぐに代替された古い規格。　https://ja.wikipedia.org/wiki/Peripheral_Component_Interconnect
ブラケットのカバー側に128MBと書いてある。　詳しい描写はされていないがLANケーブルを差して使うものだと思われる。

これがなぜ分解で手に入るかは謎。
#endregion
