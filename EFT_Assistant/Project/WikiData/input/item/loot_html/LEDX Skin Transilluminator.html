      LEDX Skin Transilluminatorのソース

      これらのキーワードがハイライトされています：
        *LEDX Skin Transilluminator [#q7852d59]
#ref(img/ledx-skin-transilluminator.jpg,nolink,left)

*概要 [#e0035447]
皮膚下の静脈の位置を正確に決定するように設計された装置。
エル・イー・ディー・エックス　（&color(Gray){もしくはレドックスなど};）と呼ばれる。

光に手をかざす（太陽に向ける）と血管が透けて見えることを利用した血管可視化装置。
迅速な注射や点滴ルート確保に役立つ。が現在の治療システムには何ら影響はしない。
#fold(&color(Black){展開};){{
トランスライト社（米国）が製造している静脈用ライト。[[公式サイト>https://www.kohkenmed.co.jp/products/kom770vlx/]]
一般的なライトでも腕の裏から当てると血管を見ることができるが、これは表側から探せるよう側射透光法（探したい場所から光らせる部分をずらす）と赤色のコールドライト（恐らく光量の割に発熱が少ないという意味）を採用し、この機器のＣ型の空いているところから注射針を刺すことができる。
